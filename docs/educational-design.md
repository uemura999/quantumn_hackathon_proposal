# Educational Design — Quantum Hackathon (高校生向け)

> 「QAOA を高校生がステップバイステップで体得する」教育コンテンツ設計

このドキュメントは、本アプリの **チュートリアル設計の意図** と、各画面で「何を見せて、何を学ばせるか」を残すためのものです。実装の単一の真実は `architecture.md` 側にあります。

---

## 1. 学習者像

- 想定: 高校生 (量子バックグラウンドなし、数学Ⅱ程度)
- 既知: 三角関数、ベクトル、確率、組合せ
- 未知: 複素数の位相、ヒルベルト空間、ハミルトニアン

→ **直接ハミルトニアンや行列式を教えない**。波と確率と組合せ爆発で語る。

---

## 2. 設計原則

| # | 原則 | 実装 |
|---|------|------|
| 1 | **1 ステップ 1 概念** | 7 ステップに分解 |
| 2 | **短い比喩 → 体験 → 機構** の 3 層 | metaphors.story / SoloSliderStep / `<details>` 機構説明 |
| 3 | **パラメータをいじったら即座に何かが変わる** | WavePanel が 60ms debounce で即時更新 |

| 6 | **専門用語にはツールチップ** | GlossaryTooltip (summary + mechanism + detail) |

## 3. 7 ステップの学習意図

### Step 0 — Welcome
- **見せる**: 道路で繋がった都市、6 配送先、渋滞色
- **教える**: 「これは物流問題で、ゴールは最短ルート」
- **言葉を残す**: 「道路」「配送先」「渋滞」

### Step 1 — Manual
- **見せる**: 6 配送先のピンを順番にクリックして手動で巡るルートを作る
- **教える**: 「人間が手で最短を当てるのは難しい」(720 通りの存在感)
- **言葉を残す**: 「順位 = N 位 / 720」「最短との差 = +X」

### Step 2 — Candidates
- **見せる**: γ=β=0 で実行 → 全 720 ルートが薄く同時に光る + 候補バーが全て同じ高さ
- **教える**: 「最初は全候補を同じくらいの強さで並べる」
- **言葉を残す**: 「720 通り」「候補の強さ」

### Step 3 — 短さの好み (γ)
- **見せる**: 短さの好みスライダー + 候補バー + **InterferenceDemo**
- **教える**:
  - 0 → 何の好みも伝えていない、候補は均等
  - 1.0 → 短い候補が伸びやすい
  - 2.8 → 強すぎて別候補にも強さが流れる
- **言葉を残す**: 「短い道に印をつける」

### Step 4 — 混ぜる強さ (β)
- **見せる**: 混ぜる強さスライダー + 候補バー
- **教える**:
  - 0 → 候補どうしが動かず差が出にくい
  - 0.4 → 短い候補の強さが見えやすい
  - 1.4 → 混ぜすぎて差がぼやける
- **言葉を残す**: 「候補を混ぜる」

### Step 5 — Reps
- **見せる**: 考え直す回数スライダー + **LayerReplay** (タイムライン再生)
- **教える**: 「印をつける → 混ぜる」を何回くり返すか
- **言葉を残す**: 「考え直す回数」

### Step 6 — Recap + Traffic
- **見せる**: 3 つのパラメータカード (γ/β/p) + 渋滞モジュール
- **教える**: 渋滞があると最短ルートが変わる現象 → 「現実物流での意味」
- **誘導**: チャレンジ画面へ

## 4. なぜ「候補バー」を常時表示にしたか

ユーザー (高校生本人) からの指摘:
- 「どんなにパラメータを調整しても結果が変わらないように見える」
- 「波とは何か、どうやってぼんやりからルートを決めているのかわからない」

→ 3D シーンだけでは「波の山と谷」が直感に届かない。
→ 候補の強さを **バー** (棒グラフ) という小学生でも読める表現に直し、3D シーンの隣に常駐させた。
→ スライダーを動かす瞬間にバー高が変わる **因果関係の即時性** が、初学者の "あ、これが効いてるんだ" を作る。

---

## 5. γ=2.8 で何を見せたかったか

「強くするほど良くなる」という素朴な期待を一度壊す。

- γ ≤ π では位相回転が単調 → 短ルートが強調される
- γ > π では長ルートの位相が一周して短ルートと同位相になる
- 結果: 1 位候補の選ばれやすさが変わる/別候補にも強さが流れる

これは量子の **非線形さ** と **適切なゾーンの存在** を体で理解する核心。
「ちょうど良い」を探すこと自体が QAOA の研究テーマ(変分量子計算)に通じる。

## 6. アクセシビリティ

- `prefers-reduced-motion: reduce` → ExecutionNarration を 50ms に短縮、フォグのパルス停止
- WavePanel/InterferenceDemo/LayerReplay は **キーボード操作可能** (スライダー / details / button)
- 色の意味は必ず**形 + テキスト**でも伝える (候補バーの高さ = ラベルの確率 % と二重符号化)

## 7. 用語の 3 層構造 (`glossary.ts`)

```ts
{
  summary: '比喩レベル (小中学生にもわかる)',
  mechanism: '機構レベル (高校生がわかる)',
  detail: '形式レベル (大学初年級向け)',
}
```

`<GlossaryTooltip>` がこの 3 層を全部表示する。読み飛ばしやすい順に並ぶ。
